# apps/

The `apps/` directory contains runnable service packages that expose modular-accounting functionality in different runtimes.

- `api/` – FastAPI application modules, persistence models, and service layers.
- `extensions/` – Extension loader and shared plugin contracts.
- `modular_accounting/` – Domain models, application services, and adapter interfaces.
- `observability/` – Structured logging, tracing, and health instrumentation.
- `web/` – Streamlit experience for interactive exploration.

Refer to [docs/architecture/overview.md](../docs/architecture/overview.md) for diagrams and module boundaries.
